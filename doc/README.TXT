■このソフトは

  Susie32 で Pi 形式の画像ファイルを表示させるための Plug-in です。

  Susie の作者の Takechin さんが公開している Susie 32bit plug-in
  library の中にも Pi Plug-in が含まれていますが、この純正版の Pi
  Plug-in には希に画像が崩れるバグがあるので、その代用品として作った
  ものです。

  純正版の Pi Plug-in と比べて、以下のような特徴があります。

    ＊ファイルサイズが小さい。現在出回っている Susie Plug-in の
      中でも、最も小さい部類に入ると思います。

    ＊純正版と比べて、10〜30% 程度高速に動作します。

■動作可能な OS

  Win32 のプログラムを動作させることのできる OS (Windows95/98/Me/NT/
  2000/XP など) が必要です。

  上記の OS の他に、Windows 3.1 + Win32s での動作にも正式に対応して
  います。


■インストール

  Susie がインストールされている場合は Susie のあるフォルダに、そう
  でない場合は Windows の System フォルダに、IFPI2.SPI をコピーして
  ください。基本的に、あとは何もする必要はありません。Susie 以外の
  ソフトの場合で、プラグインファイルのインストール場所が指定されて
  いる場合は、その指定に従ってください。

  純正版の Pi Plug-in が IFPI.SPI というファイル名なので、この非純正
  版は IFPI2.SPI という名前になっていますが、IFPI.SPI に名前を変えて
  純正版と置き換えても OK です。純正/非純正の区別は、ファイルの
  プロパティや Susie のバージョン情報などで確認できます。

  純正版の Pi Plug-in と共存させる場合は、プラグインを使用するソフト
  の方で、純正版の Pi Plug-in を使わないように設定してください。
  Susie の場合は「設定」→「プラグイン」で設定できます。


■ Pi 形式 とは

  Pi 形式は、1990年頃に 柳沢 明 氏によって提唱された 16/256色画像用
  のファイル形式です。

  提唱された当時から 16/256色画像用ファイルフォーマットとしては、
  海外では GIF、国内では MAG が主流になっていたため、実際にはあまり
  普及していない比較的マイナーな画像形式です。しかし画像圧縮の性能は、
  提唱から10年以上経た現在でも見劣りしないもので、私がテストした範囲
  ではほとんどの画像で GIF を上回る圧縮率を示しました。画像内容にも
  よりますが、平均的には PNG とほぼ同等かそれよりもやや高い圧縮率を
  示すようです(ただし PNG には、伝送エラーを検出するためのエラー
  チェック情報も含まれているという違いがある)。

  規則的なパターンや単一色の部分が多い画像に対しては、より圧縮率が
  高くなる性質があります。極端な例では、4000x3000,16色 で全面単一色
  の画像(BMP形式では5.72MB) を Pi 形式にすると、77バイトにまで小さく
  なりました(そのうち 66バイトがヘッダとパレット情報)。逆に、誤差
  拡散やディザなどの疑似階調処理が強くかかった画像(見た目でザラザラ
  した感じの画像)では、GIF や PNG よりもファイルサイズが大きくなって
  しまうようです。

  GIF や PNG では、LZW法 や deflate (LZ77+Huffman法) という汎用的な
  データ圧縮アルゴリズムを使っているのに対し、Pi 形式では画像データ
  に特化したシンプルな圧縮アルゴリズムを使っています。圧縮法が比較的
  シンプルなため、圧縮速度が比較的高速です。

  Pi 形式を書き出せる画像ファイルコンバータでは、てんか☆ さんの
  MG.EXE (MS-DOS版ソフト) が有名です。この Plug-in の動作試験と
  Pi 形式の性能試験も、この MG が出力したファイルで行なっています。


■このプラグインが返すエラーコード

  このプラグインは、Takechin さんが公開している Susie 32bit plug-in
  library に収められているプラグインと同様に、ファイルを読んでいる
  途中でファイル終端を検出した場合にはエラーコード10を返します。

  ・エラーコード
     0 : 正常終了
    -1 : その機能はインプリメントされていない
     1 : コールバック関数が非0を返したので展開を中止した
     2 : 未知のフォーマット
     3 : データが壊れている
     4 : メモリーが確保出来ない
     5 : メモリーエラー (Lock出来ない、等)
     6 : ファイルリードエラー
     7 : 窓が開けない (非公開)
     8 : 内部エラー
     9 : ファイル書き込みエラー (非公開)
    10 : ファイル終端を検出 (非公開)

■改版履歴

  ●Ver.0.10 (1999/03/20)

    ・正式版(初版)公開。

  ●Ver.0.11 (1999/04/20)

    ・ファイル変換ソフトの MG などが出力する拡張ヘッダ情報に対応
      させた。拡張情報として格納されている情報のうち、画像の表示

      意味はなし(^^;。

    ・Susie 以外の一部の Susie32 Plug-in 対応ソフトで動作しない
      バグを修正した。

    ・ファイルを読んでいる途中でファイル終端を検出した場合は、
      エラーコード10(ファイル終端/End of File)を返すように仕様変更
      した。

    ・ソフトの配布条件をすこし見直し・修正した。

  ●Ver.0.12 (1999/09/24)

    ・若干の内部的な手直しを行なっただけで、機能的な追加・変更は
      なし。

  ●Ver.0.13 (2000/05/30)

    ・プログラムを大幅修正して、冗長なコードを無くすなどの整理を
      行なった。その結果、ファイルサイズが 1KB も減って 9.5KB に
      なった。

  ●Ver.0.14 (2000/10/25)

    ・Windows 3.1 + Win32s (V.1.30c) での動作を確認、正式対応した。

    ・その他、プログラムを一部修正。

  ●Ver.0.20 (2004/09/07)

    ・Pi ファイルの展開コードを大幅に改良した。その結果、以前の
      バージョンに比べて、15〜30% 程度高速化した。

    ・横幅が2pixel以下の画像に対応した。ただし、正規の方法ではなく、
      他のソフトと互換がとれる方法を採用している。

  ●Ver.0.21 (2006/02/15)

    ・コードの最適化オプションを、今までの Pentium から Pentium II
      に変更した。その結果、Pentium II 以降の CPU においては以前と
      比べて 5% 程度高速化した。
